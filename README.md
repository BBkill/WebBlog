# Some features
USER ( use MySQL and SQL )
  - Login
  - Register
  - Change password
  - Active account
  - Get otp active account
  - Reset forget password
  - Remove account with ADMIN_ROLE
 
BLOG ( use MongoDB and NoSQL )
  - Create Blog
  - Edit Blog
  - Rate Blog
  - Comment Blog
  - Delete Blog
  - Find blog by username, title, content
